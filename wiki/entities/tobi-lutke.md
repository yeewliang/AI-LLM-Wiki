# Tobi Lütke

## Overview
Founder and CEO of Shopify. Programmer by background. Active builder of personal developer tools, several of which are released as small open-source projects from his GitHub account.

## Why this entity matters to AI
Built and released `qmd`, a hybrid BM25 + vector search engine over local markdown files exposed as an MCP server. `qmd` is positioned as the navigation layer for [[llm-wiki-pattern]] implementations once the wiki grows past the practical limit of a single `index.md` file. Notable as an example of a non-AI-native CEO directly producing AI-adjacent tooling.

## Key works / outputs
- Shopify (2006–) — e-commerce platform
- `qmd` (github.com/tobi/qmd) — local markdown search using BM25 + vector hybrid; MCP server interface; cited in 2026 LLM Wiki ecosystem coverage as the answer to the index.md scale ceiling
